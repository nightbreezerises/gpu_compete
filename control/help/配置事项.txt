如何删除前端旧产物，增加新产物：
1. 删除旧的前端构建产物（如 dist/ 目录）
2. 删除 node_modules 依赖
3. 重新安装依赖：npm install
4. 重新构建：npm run build
5. 重启后端服务

ESC 键功能：
- 所有输入框支持 ESC 键清空当前输入
- 数字输入框清空为空字符串（可重新输入新数字）
- 文本输入框清空为空字符串

YAML 格式保持：
- 使用字符串替换方案保留注释
- 数组保持行内格式 [0,1,2,3]
- 值更新时保留原有注释
- 不再创建备份文件，由 Git 管理版本
